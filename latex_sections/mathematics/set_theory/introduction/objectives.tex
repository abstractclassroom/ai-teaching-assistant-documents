\begin{itemize}
  \item Understand the convenience of referencing objects by encapsulating them in sets.
  \item Understand the notation for indicating whether an object is an element of a set.
  \item Know the benefits and caveats associated with the three primary ways of declaring/representing sets.
  \item Be able to convert sets from one form to another.
  \item Be aware that sets can contain other sets.
\end{itemize}

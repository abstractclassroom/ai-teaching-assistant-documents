\begin{itemize}
 \item Students are not familiar with the famous infinite sets or their notation. For example, $\mathbb{N}$, $\mathbb{Z}$, $\mathbb{Q}$, $\mathbb{R}$, and $\mathbb{C}$.
 \item When providing examples or exercises for this lecture, it is fine to use \mathbb{W} or \mathbb{N} to represent the set of whole numbers or natural numbers, respectively. However, it is important that you define these sets first before using them. For example, you can define $\mathbb{W} = \{0,1,2,3,\ldots\}$ and $\mathbb{N} = \{1,2,3,\ldots\}$.
 \item Math symbols are difficult to impossible to type using a keyboard.  When presenting the student with questions that involve latex symbols like $\in$ and $\notin$ or famous infinite set symbols, please provide a multiple choice question where the symbols presented and ask the student which one should be selected.
 \item There are times when students need to demonstrate they know how to write set notation.  Encourage them to write the solution on paper and take a picture of it and upload it to the prompt. Laptops and cell phones often have built-in cameras.
 \item When making it clear that sets do not have duplicates, remind students that a set written with duplicates is not technically incorrect, but it does not represent proper form. For example, the set $\{1,2,3,3\}$ is not technically incorrect, but it is not in proper form. The correct form would be $\{1,2,3\}$.
 \item This lecture is intended to only introduce sets.  Future lectures will cover more advanced topics like subsets, power sets, unions, intersections, and set differences.  It is important to keep the focus on the basics of sets in this lecture and not get too far ahead with more advanced topics.
 \item Do provide examples of sets that are infinite.  Roster form and set-builder form would be good places to provide examples of infinite sets. Make a brief comment about the difference between infinite and finite sets.
 \item Please make sure the above points and suggestions are included in appropriate places.
\end{itemize}

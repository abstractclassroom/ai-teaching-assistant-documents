\begin{itemize}
    \item What is the purpose of a set?
    \item What are the three primary ways of declaring/representing sets?
    \item What are the the benefits and problems of representing a set using sentence form?  What are some of the problems associated with using sentence form?
    \item When is it not appropriate to use roster form.
    \item What types of sets are good candidates for set-builder form.
    \item What is the notation for indicating whether an object is an element of a set?
    \item Convert the set $\{2k^2 \mid k \in \{1,2,3,4,5\}\}$ to roster form.
    \item Introduce the term perfect square and ask the student to convert the set $\{1,4,9,16,25,...\}$ to set builder form.
    \item Convert the set $\{1,2,3,\ldots,100\}$ to sentence form.
    \item Give an example of a set that contains other sets.
\end{itemize}
